# 页面拓扑清单

## 业务模块划分

- 认证模块：`/login`
- 工作台模块：`/app/dashboard`
- 历史模块：`/app/history`
- Wiki 模块：`/app/wiki/:taskId`
- 兜底模块：`*`（404）

## 页面跳转关系

- `/` -> 读取上次访问路径；存在则恢复，否则跳到 `/app/dashboard`
- `/login` -> 登录成功后跳转 `next` 参数或 `/app/dashboard`
- `/app/dashboard` -> 任务完成后编程式跳转 `/app/wiki/:taskId?repo=...`
- `/app/history` -> 点击任务跳转 `/app/wiki/:taskId`；点击聊天跳转 `/app/wiki/:taskId?chatId=...`
- 任意未知路径 -> `*` -> 404 页面

## 守卫拓扑

- 全局守卫 `GlobalRouteGuard`：记录路由埋点与最近访问路径缓存
- 路由独享守卫 `RoutePermissionGuard`：按角色约束路由访问
- 登录守卫 `AuthGuard`：未登录统一重定向到 `/login`
- 组件内守卫 `ComponentDataGuard`：Wiki 视图在任务数据未就绪时阻断渲染

## 布局与路由视图映射

- 根布局 `RootLayoutView`：背景与全局容器，承载一级 `Outlet`
- 业务布局 `AppLayout`：`Header + Sidebar + Footer + Outlet`
- 业务页面通过 `Outlet` 注入，形成嵌套路由布局
